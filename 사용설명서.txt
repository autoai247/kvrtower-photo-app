================================================================
  카메라 영수증 프린터 - 사용 설명서
================================================================

* 인스타 해시태그 프린터와 완전히 별개로 동작하는 단독 프로그램입니다.
* 같은 PC에 동시에 깔아두고 써도 충돌하지 않습니다.

[ 무엇을 하는 프로그램? ]
- 카메라(USB 웹캠 / 노트북 내장 카메라)의 영상을 실시간으로 보여주고,
- "사진 찍어서 인쇄" 버튼을 누르면 그 순간의 화면을 캡처해서
- 엡손 영수증 프린터(80mm, ESC/POS)로 즉시 인쇄합니다.

----------------------------------------------------------------
1. 설치 (한 번만)
----------------------------------------------------------------
  install.bat 더블클릭 → UAC 창에서 "예"

  자동으로 처리되는 것:
   - Python 설치 (없을 때만)
   - 필요한 라이브러리(Pillow, OpenCV, pywin32) 설치
   - 실행파일(.exe) 빌드
   - 바탕화면 바로가기 생성

  ※ 엡손 영수증 프린터 드라이버는 별도로 설치되어 있어야 합니다.
    인스타 프린터에서 이미 쓰던 프린터라면 추가 설치 불필요.

----------------------------------------------------------------
2. 실행
----------------------------------------------------------------
  바탕화면 [카메라영수증프린터] 아이콘 더블클릭
  또는 dist\카메라영수증프린터.exe

  ※ 개발자 모드(코드 수정 테스트): run.bat 실행

----------------------------------------------------------------
3. 화면 보는 법
----------------------------------------------------------------
  [왼쪽 큰 영역]   카메라 라이브 미리보기
                  파란 외곽선 안쪽이 실제로 인쇄되는 영역
                  (영수증 폭 비율에 맞춰 자동 크롭)

  [오른쪽 패널]    카메라 선택 (USB 여러 대일 때 0/1/2 변경)
                  프린터 선택 (윈도우에 등록된 프린터 목록)
                  상단 제목  (예: "K-Culture VR Tower")
                  하단 날짜·시간 체크박스
                  세로 비율  (정사각 / 3:4 / 9:16)

  [큰 파란 버튼]   사진 찍어서 인쇄
                  → 누르는 순간의 영상 캡처 + 즉시 출력

  키보드 단축키: Space 또는 Enter

----------------------------------------------------------------
4. 자주 묻는 것
----------------------------------------------------------------
  Q. 카메라가 까맣게 나옵니다.
    → 오른쪽 패널의 "카메라"를 0 → 1 → 2 로 바꿔보세요.
       (PC에 카메라가 여러 개 있을 때)
       또는 다른 프로그램(줌·OBS 등)이 카메라를 점유 중이면
       그 프로그램을 끄세요.

  Q. 인쇄가 안 됩니다.
    → 오른쪽 "프린터" 목록에서 엡손 영수증 프린터를 직접 선택.
       (빈칸이면 윈도우 기본 프린터로 출력)
    → 프린터 전원/USB/용지 확인.

  Q. 인쇄가 너무 진하거나 흐립니다.
    → 조명이 너무 어둡거나 너무 강하면 영수증의 흑백 디더링이
       지저분해집니다. 부드러운 실내 조명에서 잘 나옵니다.

  Q. 같은 버튼을 빠르게 여러 번 누르면?
    → 1초 잠금이 걸려 있어 중복 출력은 막힙니다.

  Q. 설정은 어디 저장되나요?
    → 사용자 폴더의 .camera_printer.json
       (카메라 번호, 프린터 이름, 제목 등을 기억함)

----------------------------------------------------------------
5. 문제 발생 시
----------------------------------------------------------------
  - 한 번 닫고 다시 실행해 보세요. (대부분 해결됨)
  - 그래도 안되면 install.bat 을 다시 한 번 실행.
